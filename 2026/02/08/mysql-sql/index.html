数据库基本命令学习

基本命令：

  1. drop database if exists DB； 如果存在DB数据库，删除
  2. create database if exists DB default character set utf—8 collate utf8_general_cl; 创建一个DB数据库， 字符集 utf-8 排序规则 utf8_general_cl
  3. show tables: 返回所有表名
  4. turncate Table： 清空Table表， 与drop不同， 不会删除表结构， 索引自增id。
  5. create table Table Values engine = innoDB default charset=utf8 comment = “备注” ；创建表 引擎innoDB ，
  6. select table_comment from information_schema.tables where table_name = Table; 获取Table的备注
  7. show full columns from Table; 获取Table 所有字段及定义， full 返回数据更多。
  8. show index from Table; 返回table 所有索引
    1. non_unique: 0 唯一索引 1：非
    2. key_name:索引名称 parimary | idx_user 等
    3. collation：排序类型 升序 | 降序
    4. Cardinality：索引唯一值估算数： 这个分析 索引效率， 值越大表示选择性越高
    5. index_type: 索引数据类型 Btree | hash等
  9. alter Table ： 修改表结构
    1. add ： 新增字段
    2. drop column ：删除字段
    3. change ： 修改字段
    4. after field ：在field字段后 添加
    5. drop primary key： 删除主键
    6. drop index : 删除索引
