行为树介绍

行为树

简介： 行为树包含逻辑节点和 行为树结构，每次寻找一个节点时候， 都从根节点出发。

构成：

​ 1.行为树有四种状态 ①成功 ②失败 ③持续运行 ④错误

​ 2.行为树分为两部分： 树集合 、节点集合

​ 树集合：归类每一个行为树AI根节点

​ 节点集合： 分为四个 组合节点、装饰节点、动作节点、条件节点

节点集合

组合节点

序列节点： 实现 and 功能 Sequence

记忆序列节点：也是and功能 只是会记忆当前运行节点 MemSeqence

选择节点：or 功能 Priority

记忆选择节点：类似 MemPriority

装饰器节点

重复节点：重复n次 返回结果 Repeat

重复到失败节点： Repeat until Failure

重复到成功节点： Repeat Until Success

反转节点： 将结果 “失败‘ ”成功“ 反转。 运行在不变

限制次数执行节点：

动作节点

执行具体的任务： 如“移动到某点”

条件节点

判断某个条件是否成立， 如“是否生命值低于1000”

使用

介绍

behavior3编译导出后 会产生一个json格式，节点主要包含如下重要字段：

id：全局唯一id

name：节点名

cotegory:节点类型 区分子树

properties：节点配置属性， 实际运行时控制循环次数、等待时间等定制参数

children：节点的子节点列表

blackboard（黑板）

存放数据的地方。 某个树的详细信息存储

运行

主要三个 回调函数 open tick close

执行顺序 open（开始前） -> tick（运行） -> close（开始后）
